# Trivia Train (trivia-visualizer)

A small React + Vite app that fetches 50 questions from [Open Trivia DB](https://opentdb.com) and provides interactive visualizations and filtering.

**Project choices**
- JavaScript + Vite
- Recharts for charts
- Multi-category selection, search bar, responsive charts
- Branding: dark/soft forest greens
- Deploy configured for GitHub Pages


## Project structure
- `src/` — source files
- `src/components` — React components (Header, Filters, Charts, QuestionList)
- `public/` — static assets (none by default)
